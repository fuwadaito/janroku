# 雀録 -JANROKU-

麻雀の成績を記録・分析する、1ファイル完結のWebアプリです。半荘ごとに局の内容を入力すると、最終点数・順位・収支を自動で計算し、メンバーごとの成績をまとめます。

インストール不要・ビルド不要。`index.html` をブラウザで開くだけで動きます。

## 特徴

- **局単位で記録** — 誰がどの局でロン／ツモ／流局したか、点数、リーチ、テンパイを入力
- **最終点数を自動計算** — 記録した局から持ち点を再現。ツモは各家から支払い、リーチ棒の供託や本場も自動処理（供託の残りはトップ取り）
- **収支の自動精算** — ウマ・オカ・チップまで含めてポイントを計算。素点もチップも同じ p（ポイント）で合算されるので価値が揃う（例：テンピンなら 1000点 = 100p、チップ1枚 = 100p）
- **「あなた」を設定** — メンバーの中から自分を指定でき、攻め／降りのスタンスは自分の局だけ記録
- **豊富な成績指標** — 和了率・放銃率・連対率・ラス回避率・平均順位・順位別割合・平均和了点／放銃点・攻守スタンス・収支推移グラフ・局別成績・対戦相手別成績
- **連続で記録** — 保存後に「同じ設定でもう1半荘」で、日付・メンバー・レートを引き継いで次の半荘へ（席を1つ回すことも可能）
- **スマホ・iPad対応** — レスポンシブ設計。ホーム画面に追加すればアプリのように使える
- **データはブラウザに保存** — `localStorage` に記録が残るので、閉じても消えない

## 使い方

1. `index.html` をブラウザで開く
2. 「メンバー」タブで一緒に打つ人を登録（最初に登録した人が「あなた」になります）
3. 「＋ 対局を記録」から半荘を入力
   - **①対局設定**：日付・レート・チップレート・ウマ・オカ・席順
   - **②局の記録**：局ごとにロン／ツモ／流局、和了者・放銃者・点数・リーチ・スタンスを入力
   - **③終局結果**：局データから点数・順位・収支が自動計算される
4. 「成績」タブでメンバーごとの分析を確認

局を全部入力しない場合は、③の「最終点数を手入力する」で直接入力もできます。

## 対応ルール

- 4人打ち・半荘（東南戦）
- レート：テンピン（1000点=100p）／テンゴ（50p）／テンサン（30p）／テンイチ（10p）
- ウマ：なし／5-10／10-20／10-30／20-30
- オカ：あり（25000持ち30000返し）／なし
- チップ：1枚あたりのポイントを設定

### 現在の制約

ダブロン・流し満貫・途中流局といった特殊なケースは局入力のフォームでは表現できません。その半荘だけ「最終点数を手入力する」で調整してください。

## 技術

- HTML / CSS / JavaScript のみ（フレームワーク・ビルドツール不使用）
- 外部ライブラリなし
- データ保存：ブラウザの `localStorage`

## デプロイ

静的ファイル1つなので、そのままホスティングできます。

- **Netlify**：GitHubリポジトリを連携するか、[Netlify Drop](https://app.netlify.com/drop) に `index.html` をドロップ。ビルドコマンド・公開ディレクトリは空欄でOK
- **GitHub Pages**：リポジトリの Settings → Pages から公開

## 注意

記録は開いた端末のブラウザ（`localStorage`）に保存されます。URLを共有しても記録は共有されず、端末・ブラウザごとに別のデータになります。個人用の記録ツールとしての利用を想定しています。
